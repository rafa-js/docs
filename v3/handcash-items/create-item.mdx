---
title: Create an Item
description: 'Mint new items with HandCash Items'
---

```typescript
import { HandCashMinter } from '@handcash/handcash-connect';

const minter = HandCashMinter.fromAppCredentials({ appId, appSecret, authToken });
await minter.createCollectionItemOrder({
  referencedCollection: '<COLLECTION_ID>',
  name: 'Sword of Truth',
  attributes: { season: '1', rarity: 'legendary' },
  mediaDetails: { image: { url: 'https://.../sword.png', contentType: 'image/png' } },
});
```
